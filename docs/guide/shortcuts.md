# Keyboard Shortcuts

Master these shortcuts to work faster in ModEx.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Open search / Focus search field |
| `Ctrl + N` | Create new modpack |
| `Ctrl + I` | Import mods from CurseForge |
| `Escape` | Clear selection / Close dialogs |

## Selection

| Shortcut | Action |
|----------|--------|
| `Ctrl + A` | Select all visible items |
| `Ctrl + Click` | Add/remove item from selection |
| `Shift + Click` | Select range of items |
| `Escape` | Clear current selection |

## Library View

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Focus search bar |
| `Delete` | Delete selected mods |
| `F` | Toggle favorite on selected mod |

## Modpack Editor

| Shortcut | Action |
|----------|--------|
| `Ctrl + S` | Save changes |
| `Ctrl + E` | Export modpack |
| `Delete` | Remove selected mods from modpack |

## Dialogs

| Shortcut | Action |
|----------|--------|
| `Escape` | Close dialog |
| `Enter` | Confirm / Submit |

## Navigation

Use the sidebar to navigate between views. Each view is a separate section:

- **Home** — Dashboard overview
- **Library** — All mods
- **Modpacks** — Your modpacks
- **Organize** — Folder structure
- **Stats** — Statistics
- **Sandbox** — Graph visualization
- **Guide** — Built-in documentation
- **Settings** — Configuration

## Customizing Shortcuts

Currently, keyboard shortcuts cannot be customized. This may be added in a future version.

## Tips

- Use `Ctrl + F` frequently — it works in every view
- Bulk selection with `Ctrl + Click` speeds up organization
- `Escape` is your universal "cancel" key
